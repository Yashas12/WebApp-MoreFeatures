About

Introduction

Our project provides the user with weather details in real time. This is possible through the usage of Openweathermap API.
More information about the API is provided in "Openweathermap API" option in the previous page.

The real time information is retrieved from the API using the API key and JavaScript commands. Out of the several weather attributes retrieved we use the following:
• Current temperature
• Max temperature
• Min temperature
• What it feels like
• Humidity
• Visbibility
• Wind speed and degree
• Latitude and Longitude

Usage

The Application provides the user with an easy to operate User Interface that provides real time weather data with an error margin of ±2 units.
The attributes that are retrieved are displayed in an aesthetically pleasing card shaped interface. Clicking on some of the icons takes the user to different pages in the website such as maps, wind speed information and this very page.
