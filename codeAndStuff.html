Code and Stuff

HTML

HTML in our project is used to create an aesthetically pleasing template. Through the use of bootstrap containers and rounded cards we create an User Interface where the weather values fall into.

Icons and several images are included in the HTML code to provide pleasing visuals to the application.

CSS

To stylize a plain HTML document we implement CSS in our applications. CSS is used to change the font size, weight, color, background images etc.
We imported "Raleway" font family from google fonts and used it in our CSS stylesheet.

JavaScript

The main use of JavaScript in our application is to fetch weather information from the API using our API key.

The other uses of using JavaScript in our project is temperature convertion (Kelvin to Celsius), Latitude and Longitude in Degree Decimal format to Degree Minute Second format, checking whether it's dayime or nighttime in our searched location etc.
